# Requirements:

* php >=7.2
- [x] composer

# Setup

## clone project :

```bash
git clone https://ihabkhammouma@bitbucket.org/ihabkhammouma/ibanfirst.git

composer install
```

create `.env` like `.env.example`

```
API_END_POINT=''
API_USERNAME=''
API_PASSWORD=''
```

## runing server:

```bash

php artisan serve

```
